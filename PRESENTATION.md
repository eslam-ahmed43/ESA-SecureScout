## Slide 4: Powered by Gemini 3 Pro
**Visuals**:
- Gemini Logo prominent.
- Comparison Table: "Traditional Regex Scanner" vs "Gemini 3 Pro Reasoning".
- Icon of a "Thinking Brain".

**Content**:
### Why Gemini 3 Pro?
*   **Contextual Reasoning**: Unlike regex scanners, Gemini understands *logic flow*. It knows if a variable is sanitized before use.
*   **Search Grounding**: Validates findings against real-time CVE databases (2024/2025 data).
*   **Multimodal Input**: Capable of analyzing code snippets, HTML structure, and server headers simultaneously.

**Speaker Notes**:
"This is where the magic happens. Traditional scanners look for patterns. Gemini looks for *intent*. It uses its massive context window to understand the entire application flow, reducing false positives and identifying complex logic flaws."
